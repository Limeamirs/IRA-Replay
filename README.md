# **IRA**
### **Instant Replay Assistant 🎥**

**IRA** is a **powerful instant replay recording tool** designed to capture and save your most recent gameplay moments with precision and flexibility.

Did something incredible just happen?  
Clutched the round?  
Hit a perfect shot?  

**IRA already has it saved.**

---

## ⏱ **Instant Replay Buffer**

**IRA allows you to instantly save the last 10 to 600 seconds of gameplay.**  

Fully customizable replay duration lets you decide exactly how much footage you want to keep — from quick highlights to extended moments.

---

## 🎚 **Advanced Quality Control**

IRA provides **flexible recording settings** so you can fine-tune performance and output quality:

- **Adjustable bitrate**  
- **Video quality configuration**  
- **Encoding parameter control**  
- **Performance optimization options**  
- **Balance between file size and visual quality**

Whether you're running a low-end system or a high-performance gaming rig, **IRA adapts to your setup.**
